# Shrimp Sanctuary
## The game for Secret Santa Jam 2025

### Description
Shrimp Sanctuary - is a relaxing shrimp feeding game. Your main goal is to feeding the shrimps.

You can buy different types of shrimps or wallpaper into your aquarium.

There`s no true ending, but you can get all achievements to get platina on some weird indi game, if you want to.
#
### Install instructions
On every platform the game will create saves folder next to itself, so you may want to put .exe file in the another folder and create shortcut on your desktop or in any place you want.
#### Windows
Download [ShrimpSanctuary_windows.exe](https://github.com/Gargonium/ShrimpSanctuary/releases/download/Shrimp-Sanctuary-V-1.0.0/ShrimpSanctuary_windows.exe)
#### Linux
Download [ShrimpSanctuary_linux](https://github.com/Gargonium/ShrimpSanctuary/releases/download/Shrimp-Sanctuary-V-1.0.0/ShrimpSanctuary_linux)
